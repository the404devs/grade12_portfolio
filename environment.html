Every single day, items that could be reused end up in landfills, or worse, or our streets, forests, rivers, and oceans. This waste harms the ecosystem, as well as any animal that tries to ingest it. By simply placing a plastic bottle into a recycling bin instead of just tossing it aside, you can make a big impact on the environemnt in your community.


This program, known as Recyclapp, would provide a simple interface for a user to determine whether or not an item is recyclable. By either uploading an image, or searching for keywords, the program would state whether or not the item can be recycled, and provide some additional tips, such as removing a lid or rinsing a container.


Recyclapp would make use of a database of information, containing general recycling information to be shown when the user searches with keywords. When the user uploads an image, the program will use AI to determine what the object in the image is. Once the nature of the item is determined, the information database would be accessed and return some specific information on if and how the user can recycle the item.
